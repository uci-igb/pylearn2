Scripts for training neural network on improver 2013 data, then making
predictions.

* Usage *
1) Make sure ~/improver2013 points to the github repository improver 2013, so that the dataset can be found by pylearn2 dataset file pylearn2/datsets/improver2013.py.
2) Source profile in improver2013/opt/pylearn2/profile.profile
3) Script addgauss02_width100.py will initialize a NN, train it, then print
out predictions in local directory. Hyperparameters can be easily modified
using this script.
